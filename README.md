## Setup
### Installation of Node and npm

````
curl -sL https://deb.nodesource.com/setup_4.x | sudo -E bash -
sudo apt-get install -y nodejs
````

### Installation of all dependencies

````
npm install
````

### Build bundle.js

````
webpack
````
